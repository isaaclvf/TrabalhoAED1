# Trabalho de Algoritmos e Estruturas de Dados I

Implementação de um conjunto de dados organizados por listas encadeadas.

## Requisitos

O projeto usa a biblioteca [libcsv](https://github.com/rgamble/libcsv) para leitura de arquivos csv.

No Debian/Ubuntu:

```
sudo apt install libcsv3
sudo apt install libcsv-dev
```
